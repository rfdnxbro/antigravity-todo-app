# プロジェクトナレッジ: Antigravity TODO アプリ

## 概要
React と Next.js で構築された、シンプルでポップなテーマの TODO アプリケーションです。ユーザーはタスクの追加、完了切り替え、更新、削除を行うことができます。フレンドリーで活気のあるユーザーインターフェースデザインに重点を置いています。

## 技術スタック
- **フレームワーク**: Next.js (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **状態管理**: React Hooks (`useTodos`)

## ディレクトリ構成
- `src/app`: メインアプリケーションのページとレイアウトが含まれます。
  - `page.tsx`: ホームページのメインエントリポイントおよびレイアウト。
  - `layout.tsx`: メタデータとグローバルスタイルを含むルートレイアウト。
  - `globals.css`: グローバル CSS スタイル。
- `src/components`: 再利用可能な UI コンポーネント。
  - `Header.tsx`: アプリケーションのタイトル ("TODO POP!") を表示します。
  - `AddTodoForm.tsx`: 新しいタスクを追加するためのフォーム。
  - `TodoList.tsx`: タスクのリストをレンダリングします。
  - `TodoItem.tsx`: 個々のタスクアイテムを表します。
- `src/hooks`: カスタム React フック。
  - `useTodos.ts`: Todo の状態（追加、切り替え、削除、更新）を管理します。
- `src/types`: TypeScript の型定義。
  - `todo.ts`: `Todo` インターフェースを定義します。

## 主な機能
- **タスク追加**: テキストを入力して新しいタスクを作成できます。
- **完了切り替え**: タスクを完了または未完了としてマークできます。
- **タスク削除**: リストからタスクを削除できます。
- **タスク更新**: タスクのテキストを編集できます。
- **永続化**: (注: 現在はローカルステートを使用しています。永続化の実装詳細は今後決定されます)。
